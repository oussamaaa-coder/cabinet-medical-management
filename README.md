# 🏥 Laravel Medical App

A medical management web application developed using Laravel.  
This project allows managing patients, doctors, appointments and medical records.

---

## 📌 Project Description

The **Laravel Medical App** is a web application designed to help clinics manage their daily operations.  
It provides an easy interface to manage patients, doctors, and appointments.

This project was developed as part of a backend training project.

---

## ⚙️ Technologies Used

- PHP
- Laravel
- MySQL
- Blade Templates
- HTML / CSS
- JavaScript
- Bootstrap

---

## ✨ Features

- 👨‍⚕️ Doctor management
- 🧑‍🤝‍🧑 Patient management
- 📅 Appointment scheduling
- 🗂 Medical records management
- 🔐 Authentication system
- 📊 Dashboard for administration
